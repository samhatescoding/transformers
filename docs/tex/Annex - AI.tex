\section{AI}

It's difficult to answer \textit{where} I used AI in the project, as it's been used everywhere. 
I had access to GitHub Copilot and paid Codex, and I used them to help plan tasks, draft code, reshape text, and
organize the report structure.

You see, even when I was not explicitly asking AI to produce a final sentence or code
block, the work was still AI-assisted in practice because Copilot and Codex
were continuously suggesting text and code as I wrote.  

Their suggestions were part of the normal
writing and editing workflow, not a separate step after the fact.
While I did ask AI for input about the structure, those are mostly my decisions. 
There were also parts of the work that I had to do manually because AI was not
appropriate or reliable enough for them. The dataset search is one example: I
had to read through the studies myself, exclude most of them, and decide which
ones fit the project's criteria. That selection process required judgment and
cross-checking that I did by hand.

The same was true for several benchmark annotations. AI generated most of the
distractor prompts, but I reviewed the outputs and corrected them when they
were weak, inconsistent, or did not match the benchmark. For some datasets,
such as FlyingThings3D, I had to write the prompts myself because the images
were difficult to interpret and there was not always a clear correct prompt to
start from.
