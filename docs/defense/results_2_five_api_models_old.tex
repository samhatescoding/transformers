\section{Data Source}

This report uses the frozen pre-June-11 Spanish-time snapshot in
\path{results/api_models_pre_2026_06_11_es}. The snapshot was extracted
from commit \code{96e05c3}
(\emph{Llava finetuning fix}), dated 2026-06-10 22:37:44 +0200.

The snapshot contains uneven benchmark coverage across the five API models.
For a fair five-way comparison, this report uses the balanced overlap available
for all five models: 12 Type A/Type B benchmarks. Type A corresponds to visual
question answering and visual reasoning; Type B corresponds to detection and
localization.

\section{Balanced Five-Model Results}

The balanced old comparison uses 12 benchmarks for every model: six Type A QA
benchmarks and six Type B detection benchmarks. On this overlap,
\code{gpt-4.1} has the highest mean score, followed closely by
\code{gpt-4o} and \code{gpt-5}. \code{gemini-3.5-flash} is much lower in this
old snapshot, consistent with the pre-fix Gemini result files.

\section{Pairwise Notes}

On the balanced overlap, \code{gpt-4.1} beats \code{gpt-4o} on seven
benchmarks, loses on three, and ties on two. \code{gpt-4o} and \code{gpt-5}
split evenly with four wins each and four ties. Every non-Gemini model beats
\code{gemini-3.5-flash} decisively in this old snapshot.
