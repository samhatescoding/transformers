\section{Data Source}

This report analyzes the final clean result set in
\code{results/api\_models\_final\_clean}. It contains 44 benchmarks for each of
five API models: \code{gpt-4o}, \code{gpt-4.1}, \code{gpt-5},
\code{claude-sonnet-4-5}, and \code{gemini-3.5-flash}. The result set contains
220 benchmark files and 4,345 sample results in total, with no sample-level
errors.

The figures and CSV tables were generated with:

\begin{quote}
\code{python comparison\textbackslash build\_report.py --results-dir results\textbackslash api\_models\_final\_clean --output-dir comparison\textbackslash output\textbackslash five\_api\_models\_final\_clean --bootstrap-samples 5000}
\end{quote}
